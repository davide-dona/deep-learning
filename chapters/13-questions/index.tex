\section{Questions}

\paragraph{FFNN}
\begin{enumerate}
    \item Activation functions: ReLU, Leaky ReLU, ELU, SELU, GELU, Swish, Mish. What are their pros and cons?
    \item Stochastic Gradient Descent (SGD) with Momentum.
    \item How can we regularize a neural network? (L1, L2, Dropout, BatchNorm, Data Augmentation)
\end{enumerate}

\paragraph{CNN}
\begin{enumerate}
    \item Difference between AlexNet, GoogLeNet and ResNet (key contributions and draw them).
    \item Draw and explain the inception module. What are 1x1 convolutions used for?
\end{enumerate}

\paragraph{Transformer}
\begin{enumerate}
    \item Attention formula, write down the formula and explain each term (Q, K, V, d, softmax, etc.)
    \item Differences between local attention and global attention. Draw it and write the formula.
\end{enumerate}

\paragraph{GAN}
\begin{enumerate}
    \item draw the architecture of a GAN.
    \item Write the loss function of a GAN and explain what is the problem with the original loss function? (min-max game, mode collapse, training instability).
    How can we fix it? (Switch to max-max or min-min, LSGAN)
    \item What is Mode Collapse? How can we fix it? (Minibatch discrimination, LSGAN)
    \item Can you create a GAN with some conditional information? 
\end{enumerate}
